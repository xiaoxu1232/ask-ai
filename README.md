# ask-ai

When AI Agent encounters difficult problems, automatically open AI assistant webpage to ask for help.

🔥 Pain Points
❌ AI Agent has limited model capabilities, can't solve complex problems
❌ Knowledge gaps, can't accurately answer professional questions
❌ Need deeper explanations, but AI answers are not detailed enough
❌ Complex problems require multi-round reasoning
💡 Solution
When AI Agent encounters the following, automatically open stronger AI assistants for help:

Unsolvable problems
Knowledge gaps
Need deeper explanations
Complex problems requiring multi-round thinking
User explicitly asks to consult AI
✨ Features
AI Agent automatically opens AI assistant for help when:

Encountering unsolvable problems
Knowledge gaps
Need deeper explanations
Complex problems requiring multi-round reasoning
User explicitly asks to consult AI
🌐 Supported AI Assistants
ChatGPT: https://chatgpt.com
Claude: https://claude.ai
Gemini: https://gemini.google.com
DeepSeek: https://chat.deepseek.com
Other AI assistants
📖 Usage
Trigger
User says:

"Ask AI"
"Ask ChatGPT/Claude/Gemini"
"Open AI"
"Consult AI"
Or Agent triggers automatically when facing difficulties.

Workflow
Open AI assistant webpage in browser
Wait for user login (if needed)
Type question and send
Get answer and report to user
🤖 Communication Guidelines
When Opening
"Okay, let me ask AI"
When Facing Problems
"Sorry, encountered [issue]. Need your help with [solution]"
After Getting Answer
"Got the answer from [AI name]: [summary]. Want details?"
📊 Tracking
Report results to user after each query
Log topics for future reference
🔧 Compatible Platforms
OpenClaw
Claude Desktop
Cursor
Other AI Agents with browser automation
⚠️ Requirements
User must log in to AI assistants first
Browser automation capability required
Network access to AI assistants
